❓ 常见问题

什么是 MCP（Model Context Protocol）？

MCP 是一个开放的 AI 连接协议，由 Anthropic 提出。它允许 AI 模型安全地访问外部工具和数据来源，标准化了 AI 与应用之间的交互方式。

MCP 服务是免费的吗？

当前 MCP Hub 服务免费开放使用。如需企业级定制或更高并发支持，请联系 our team。

如何自定义/添加自己的 MCP 服务？

我们的 MCP Hub 基于开源方案构建，支持自定义服务注册。您可以基于相同的架构搭建自己的 MCP Hub，或联系我们要部署专属实例。
