📖 模型配置指南

📌 第一步：获取 DeepSeek API Key

强烈推荐使用 DeepSeek 官方 API 配置小易智能体——稳定、快速、安全。

  • 官网地址：https://platform.deepseek.com
  • 注册方式：手机号/邮箱即可注册，支持微信扫码登录
  • 创建 API Key：登录后进入「API 密钥」页面，点击「创建密钥」，给密钥起个名字（如"小易智能体"）
  • 计费说明：按 token 用量计费，deepseek-chat 价格为 0.005 元/千 token，deepseek-reasoner 为 0.014 元/千 token，远低于主流大模型
  • 免费额度：新用户通常有 100 万 token 免费体验额度

🔑 第二步：进入模型配置

打开小易智能体管理后台，找到模型配置入口：

  • 登录小易智能体管理平台 → 进入「设置」或「系统配置」
  • 找到「模型服务」或「AI 模型」配置项
  • 点击「添加模型」或「编辑模型」，选择自定义 API 模式

💡 提示：如果找不到配置入口，可以在页面顶部搜索栏输入"模型"快速定位。

⚙️ 第三步：填写 API 信息

在配置页面中，按以下信息填写：

  • API 地址（Base URL）：https://api.deepseek.com/v1（已默认配置，无需修改）
  • API Key：将第一步中创建的密钥粘贴到此处
  • 模型名称：可自定义一个显示名，如"DeepSeek-V3"

⚠️ 注意：API Key 是敏感信息，请勿泄露给他人。配置完成后建议隐藏显示。

🤖 第四步：选择模型

DeepSeek 提供两个主力模型，按需选择：

  • 🗣️ deepseek-chat（推荐新手）— 通用对话模型，适合日常问答、文案写作、代码生成、翻译等。速度快、成本低、质量高，是大多数场景的首选。
  • 🧠 deepseek-reasoner（推理增强）— 深度推理模型，适合数学计算、复杂逻辑推理、学术论文、代码调试等需要"深度思考"的场景。回答质量更高，但速度略慢。
  • 🆕 deepseek-v3 — 最新版本，综合能力全面超越前代，推荐优先选择。

💡 省钱技巧：日常聊天用 deepseek-chat 即可，每次对话成本约 0.01 元。只有遇到复杂问题时再切换到 deepseek-reasoner。

✅ 第五步：测试连接

配置完成后务必先做测试，确保一切正常：

  • 点击「测试连接」按钮，系统会发送一条测试消息给 DeepSeek
  • 成功标志：显示"连接成功"或"测试通过"，且能收到 AI 回复
  • 失败排查：
    • 检查 API Key 是否正确复制（不要有多余空格）
    • 确认 API Key 未被停用或欠费
    • 检查网络是否能访问 api.deepseek.com

🎉 测试通过后，你就可以在智能体中使用 DeepSeek 模型了！
