📖 模型配置指南

📌 第一步：获取 DeepSeek API Key

强烈推荐使用 DeepSeek 官方 API 配置小易智能体——稳定、快速、安全。

  • 官网地址：https://platform.deepseek.com
  • 注册方式：手机号/邮箱即可注册，支持微信扫码登录
  • 创建 API Key：登录后进入「API 密钥」页面，点击「创建密钥」
  • 计费说明：按 token 用量计费，deepseek-chat 价格为 0.005 元/千 token

🔑 第二步：进入模型配置

打开小易智能体管理后台，找到模型配置入口：

  • 登录小易智能体管理平台 → 进入「设置」或「系统配置」
  • 找到「模型服务」或「AI 模型」配置项
  • 点击「添加模型」或「编辑模型」，选择自定义 API 模式

⚙️ 第三步：填写 API 信息

在配置页面中，按以下信息填写：

  • API 地址（Base URL）：https://api.deepseek.com/v1
  • API Key：将第一步中创建的密钥粘贴到此处
  • 模型名称：可自定义一个显示名，如"DeepSeek-V3"

🤖 第四步：选择模型

DeepSeek 提供两个主力模型，按需选择：

  • 🗣️ deepseek-chat（推荐）— 通用对话，速度快、成本低
  • 🧠 deepseek-reasoner（推理）— 深度推理，适合复杂问题
  • 🆕 deepseek-v3 — 最新版本，综合能力全面

✅ 第五步：测试连接

  • 点击「测试连接」按钮，系统会发送一条测试消息
  • 成功标志：显示"连接成功"
  • 失败排查：检查 API Key、网络、API 地址

💡 温馨提示

  • 推荐 DeepSeek 官方 API — 速度快、安全、性价比高
  • 每月费用：普通用户每月约 5 元
  • 支持所有智能体：配置后可在所有 215 个智能体中使用
