❓ 常见问题

什么是 Agent Skills？

Agent Skills 是 AI 智能体的可复用能力包，包含 SKILL.md 配置文件和相关脚本。安装后，AI 智能体可以自动加载这些技能，提升工作效率。

Skills 和 MCP 有什么区别？

Skills 侧重于提供操作指导和最佳实践（类似手册），MCP 侧重于提供 API 接口（类似工具）。两者可以互补使用。

支持哪些 AI Agent？

支持 Claude Code、Cursor、Windsurf、GitHub Copilot、Codex CLI、Gemini CLI 等 18+ 主流 AI Agent。

如何自定义/发布自己的 Skills？

创建包含 SKILL.md 的目录，然后通过 `npx skills publish` 发布到 skills.sh。我们也会在平台上收录优质 Skills。
