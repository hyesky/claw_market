# 📘 OpenClaw Agent & Soul MD 龙虾机器市场平台 PRD文档

> 基于 OpenClaw 生态，聚焦 SOUL.md / AGENTS.md / USER.md 等人格化配置文件的标准化交易与分发

---

## 1. 文档概述

| 模块 | 说明 |
|------|------|
| **产品背景** | OpenClaw 生态爆发，创作者通过 SOUL.md 等文件定义 Agent 人格与行为 [[1]][[3]]，但缺乏标准化交易渠道，优质"灵魂配置"难以商业化流通。 |
| **产品目标** | 构建 C2C/B2C 的 Agent 配置文件交易市场，实现"人格即服务"（Persona-as-a-Service），降低用户配置门槛，激励创作者持续产出高质量灵魂模板。 |
| **目标用户** | ① Soul 创作者（定义人格/工作流）② Agent 使用者/企业（采购即用型配置）③ 平台运营与审核团队。 |
| **核心指标** | GMV、配置复用率、创作者 30 日留存、下载转化率、内容审核通过率、用户配置满意度（NPS）。 |

---

## 2. 产品架构

```
┌─────────────────────────────────────┐
│  用户端（Buyer）                     │
│  ├─ 浏览/搜索 → 商品详情 → 一键导入 → 个人中心
│  └─ 支持直接对接 OpenClaw 工作区同步
├─────────────────────────────────────┤
│  创作者端（Creator）                 │
│  ├─ MD 文件上传 → 元数据配置 → 定价授权 → 数据看板
│  └─ 支持版本管理、草稿预览、批量发布
├─────────────────────────────────────┤
│  管理后台（Admin）                   │
│  ├─ 内容审核（AI+人工）→ 订单财务 → 用户权限 → 系统配置
│  └─ 支持安全风控、版权存证、违规处置
└─────────────────────────────────────┘
```

## 3. 核心功能详细说明（每项约 100 字）

| 功能模块 | 功能描述 |
|----------|----------|
| **1. 多角色注册与权限体系** | 支持邮箱/手机号/第三方登录，创作者需完成实名与 OpenClaw 工作区绑定验证。系统自动划分买家/创作者/管理员权限，提供差异化入口与数据看板，保障交易安全与配置导入合规性。 |
| **2. 智能搜索与多维筛选** | 支持按 Agent 类型（客服/开发/创意）、适用模型（Claude/GPT/KIMI）、人格标签（专业/幽默/严谨）、价格区间等多维度检索。结果页展示预览摘要与兼容性标识，提升配置匹配效率。 |
| **3. Soul 商品详情页** | 展示 SOUL.md 核心人格摘要、YAML 元数据、适用场景示例及作者介绍 [[3]]。内置"预览人格效果"沙箱、一键复制/导入按钮，支持版本对比与用户实测反馈，降低决策成本。 |
| **4. 安全结算与授权管理** | 支持单购/订阅/团队授权多种模式，集成主流支付通道。结算后自动生成授权协议（个人商用/企业商用/禁止转售），并绑定用户 OpenClaw 工作区 ID，实现配置溯源与防扩散。 |
| **5. 创作者上传与版本管理** | 提供标准化上传表单，要求填写 SOUL.md/AGENTS.md 等文件、模型兼容性、人格描述与定价策略 [[20]]。支持语义版本控制（semver）、草稿预览、批量上下架，后台实时同步审核状态。 |
| **6. 自动化分账与提现** | 系统按订单自动分账，扣除平台佣金后计入创作者余额。支持设置提现阈值、绑定银行卡或加密货币钱包，每月生成税务合规对账单，财务流程全自动化并支持多币种结算。 |
| **7. 评价反馈与人格问答社区** | 买家导入配置后可对人格匹配度、行为稳定性、作者响应进行星级评价与图文反馈。详情页增设技术问答区，创作者可解答配置调优问题，沉淀社区知识并增强复购信任。 |
| **8. 创作者数据分析看板** | 可视化展示下载量趋势、热门人格标签分布、转化率及收入明细。提供竞品对标建议与用户画像分析，辅助创作者优化人格设定与定价策略，实现数据驱动的精细化运营。 |
| **9. 内容审核与安全风控** | 结合静态代码扫描与 AI 语义分析，对上传 MD 文件实施敏感指令过滤、权限越界检测与人格合规评级 [[18]]。建立违规扣分与下架机制，自动拦截恶意 Prompt Injection 配置。 |
| **10. 个性化推荐与流量分发** | 基于用户工作区配置、浏览历史与人格偏好，构建个性化推荐引擎。首页动态展示"新人格首发""企业热销榜""场景化推荐"，提升长尾 Soul 内容曝光与整体 GMV。 |

## 4. 非功能性需求

| 类别 | 要求 |
|------|------|
| **性能** | 首屏加载≤1.5s，MD 文件解析≤200ms，支持万级并发下载请求，大文件（≤50MB）分片传输 [[20]]。 |
| **安全** | 全链路 HTTPS，配置文件沙箱预览防执行，敏感操作二次验证，防爬虫与 WAF 防护，关键数据加密存储。 |
| **兼容性** | 适配 OpenClaw 官方客户端及主流聊天插件（Telegram/WhatsApp/Discord），支持 Web/PWA/CLI 多端访问。 |
| **可用性** | 核心交易链路 SLA≥99.95%，异常自动降级与监控告警，支持灰度发布与配置回滚，保障创作者与买家体验。 |

| **V2.5** | OpenClaw 插件市场集成、人格进化追踪（AI 辅助优化）、创作者成长体系、跨平台配置同步 |

## 6. 附录

### 🔹 技术规格参考
- **文件格式**：Markdown + 可选 YAML frontmatter，`description` 字段用于 UI 摘要展示 [[20]]
- **文件限制**：单 Bundle ≤50MB，仅 SOUL.md 参与 Embedding 索引 [[20]]
- **版本控制**：Semver 语义版本，`latest` 标签指向稳定版，支持回滚
- **Slug 规范**：小写字母+数字+短横线，URL-safe：`^[a-z0-9][a-z0-9-]*$` [[20]]

### 🔹 埋点清单（关键路径）
```
曝光：商品列表页、推荐位、搜索结果

转化：支付成功、配置导入成功、工作区同步完成
留存：7 日/30 日复购、创作者内容更新频率
安全：敏感词拦截次数、审核驳回原因分布
```
