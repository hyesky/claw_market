# 🛡 Objection Handler — Sales Rebuttal Engine

> Real-time rebuttals and talk tracks that help sales reps navigate objections with confidence.

## Overview
Objection Handler classifies sales objections by type (price, timing, competition, authority, need), identifies the root concern behind surface-level pushback, and provides multiple response options with coaching notes. It helps reps respond naturally, not from a script.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/objection-handler/agent
cp SOUL.md ~/.openclaw/agents/objection-handler/agent/
openclaw agents add objection-handler --workspace ~/.openclaw/agents/objection-handler
```

## Use Cases
| Request | Output |
|---------|--------|
| "They said your price is too high" | 3 response options with coaching notes |
| "We're happy with our current solution" | Status-quo rebuttals with discovery questions |
| "I need to talk to my boss" | Authority-chain talk tracks to advance the deal |
| "Build an objection playbook for enterprise sales" | Full playbook organized by objection category |
