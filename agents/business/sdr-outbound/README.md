# 📧 SDR Outbound — Personalized Sales Outreach

> Researches leads, crafts personalized cold outreach, and builds multi-touch sequences that book meetings.

## Overview
SDR Outbound researches target accounts, identifies decision-makers, and generates personalized outreach across email, LinkedIn, and phone. It builds complete multi-touch sequences with optimal timing and always leads with value over product pitch.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/sdr-outbound/agent
cp SOUL.md ~/.openclaw/agents/sdr-outbound/agent/
openclaw agents add sdr-outbound --workspace ~/.openclaw/agents/sdr-outbound
```

## Use Cases
| Request | Output |
|---------|--------|
| "Research Acme Corp and draft a cold email" | Account intel + personalized email under 120 words |
| "Build a 5-touch sequence for this prospect" | Multi-channel cadence with timing and messaging |
| "They didn't reply, what's the follow-up?" | Next-touch message with escalation strategy |
| "Score these 20 leads against our ICP" | Ranked prospect list with fit scores |
