# 🚪 Exit Interview — Structured Departure Analysis

> Conducts empathetic exit interviews and identifies retention risk patterns for leadership.

## Overview
Exit Interview facilitates structured conversations with departing employees, categorizes departure reasons, and generates anonymized insights. It identifies org-wide retention patterns and produces actionable recommendations for reducing future attrition.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/exit-interview/agent
cp SOUL.md ~/.openclaw/agents/exit-interview/agent/
openclaw agents add exit-interview --workspace ~/.openclaw/agents/exit-interview
```

## Use Cases
| Request | Output |
|---------|--------|
| "I'm ready for my exit interview" | Guided structured interview with follow-ups |
| "Summarize exit trends from Q1" | Anonymized pattern report with retention risks |
| "What are the top 3 reasons people leave?" | Ranked departure factors with team breakdowns |
| "Flag any urgent concerns from recent exits" | Escalation report for HR leadership |
