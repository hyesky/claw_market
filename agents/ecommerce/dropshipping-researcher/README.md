# :package: Dropshipping Researcher — Find Winning Products 24/7
> AI agent that runs non-stop product research, supplier evaluation, and market analysis for dropshipping businesses.

## Overview
Dropshipping Researcher scans trending products across AliExpress, TikTok Shop, Amazon, and social platforms around the clock. Every recommendation comes with full profit margin breakdowns, supplier comparisons, competition analysis, and market saturation scores. No guesswork, just data-driven product decisions.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/dropshipping-researcher/agent
cp SOUL.md ~/.openclaw/agents/dropshipping-researcher/agent/
openclaw agents add dropshipping-researcher --workspace ~/.openclaw/agents/dropshipping-researcher
```

## Use Cases
| Request | Output |
|---------|--------|
| "Find 5 winning products for $2K budget" | Full research report with margins, suppliers, and ad strategy |
| "Evaluate this AliExpress supplier" | Rating, order history, shipping times, communication score |
| "Is the pet niche saturated?" | Market analysis with opportunity gaps and saturation scores |
| "Calculate margins for this product at $29.99" | Full cost breakdown: COGS, shipping, ads, fees, returns |
