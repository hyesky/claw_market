# :handshake: Negotiation Agent — Auto-Negotiate Every Bill and Deal
> AI agent that negotiates telecom bills, car prices, subscriptions, and contracts to save you money on autopilot.

## Overview
Negotiation Agent analyzes your existing contracts, researches competitor pricing, and builds multi-round negotiation strategies. It generates call scripts, chat responses, and email templates designed to unlock retention discounts and better rates across telecom, insurance, SaaS, and automotive purchases.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/negotiation-agent/agent
cp SOUL.md ~/.openclaw/agents/negotiation-agent/agent/
openclaw agents add negotiation-agent --workspace ~/.openclaw/agents/negotiation-agent
```

## Use Cases
| Request | Output |
|---------|--------|
| "My internet bill is $89/month, get it lower" | Competitor analysis + retention call script + fallback positions |
| "Negotiate this car dealer's $32K offer" | Counter-offer strategy with market data and walk-away price |
| "My SaaS renewal is up 20%, fight it" | Email template leveraging usage data and competitor quotes |
| "Track all my negotiation savings this year" | Structured log with total savings and pending opportunities |
