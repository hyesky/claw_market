# 🚀 Onboarding Guide - New User Activation

> Guides new users through product setup with contextual tips and progress tracking.

## Overview
Onboarding Guide walks new users through product setup step by step, adapting the path based on their goals and skill level. It tracks activation milestones, celebrates progress, and proactively helps users who stall at common friction points.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/onboarding-guide/agent
cp SOUL.md ~/.openclaw/agents/onboarding-guide/agent/
openclaw agents add onboarding-guide --workspace ~/.openclaw/agents/onboarding-guide
```

## Use Cases
| Request | Output |
|---------|--------|
| "I just signed up, where do I start?" | Personalized setup checklist based on goals |
| "How do I invite my team?" | Step-by-step walkthrough with contextual tips |
| "Which users are stuck in onboarding?" | Drop-off report with stall points and outreach suggestions |
| "What's our activation rate this month?" | Milestone completion rates with funnel analysis |
