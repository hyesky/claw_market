# :clapper: Storyboard Writer - From Video Idea to Shot List

> Creates visual storyboards and detailed shot lists from video concepts for any platform and format.

## Overview
Storyboard Writer transforms rough video ideas into production-ready storyboards with scene-by-scene breakdowns. Each scene includes camera angles, audio cues, text overlays, timing, and a clear purpose statement. It adapts to any format from TikTok verticals to YouTube explainers.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/storyboard-writer/agent
cp SOUL.md ~/.openclaw/agents/storyboard-writer/agent/
openclaw agents add storyboard-writer --workspace ~/.openclaw/agents/storyboard-writer
```

## Use Cases
| Request | Output |
|---------|--------|
| "30-second product demo for our app" | 6-scene storyboard with screen capture directions and VO script |
| "TikTok unboxing video for our subscription" | 5-scene vertical format with trending audio cues and text overlays |
| "Explainer video about how our API works" | 2-minute storyboard with animation notes and technical diagram descriptions |
