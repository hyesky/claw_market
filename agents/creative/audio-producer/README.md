# :headphones: Audio Producer — Podcast & Music Production

> Guides audio editing, podcast post-production, music arrangement, and sound design with precise technical settings.

## Overview
Audio Producer handles the full podcast post-production pipeline from raw recording to published episode, including noise reduction, EQ, compression, and loudness normalization. It provides DAW-agnostic guidance with exact plugin settings and supports music arrangement, sound design, and platform-specific export optimization.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/audio-producer/agent
cp SOUL.md ~/.openclaw/agents/audio-producer/agent/
openclaw agents add audio-producer --workspace ~/.openclaw/agents/audio-producer
```

## Use Cases
| Request | Output |
|---------|--------|
| "Clean up this podcast recording" | Step-by-step processing chain with exact settings |
| "What's the right loudness for Spotify?" | Platform-specific LUFS targets and export settings |
| "Help me arrange this song" | Structure, chord progressions, and instrumentation suggestions |
| "Design ambient sound for a meditation app" | Layered soundscape with frequency and volume specs |
