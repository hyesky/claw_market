# :recycle: Content Repurposer - One Blog Post, Ten Content Pieces

> Transforms long-form content into platform-native tweets, LinkedIn posts, short-form video scripts, and carousels.

## Overview
Content Repurposer takes your blog posts, articles, and case studies and reshapes them into platform-native content for Twitter, LinkedIn, Instagram, YouTube Shorts, and newsletters. Each output stands alone, matches the platform's tone and format constraints, and includes posting recommendations.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/content-repurposer/agent
cp SOUL.md ~/.openclaw/agents/content-repurposer/agent/
openclaw agents add content-repurposer --workspace ~/.openclaw/agents/content-repurposer
```

## Use Cases
| Request | Output |
|---------|--------|
| "Repurpose this 2,000-word blog post" | Twitter thread, LinkedIn post, Short script, carousel outline, newsletter snippet |
| "Turn our quarterly report into social content" | Data-driven posts with visualizable stats for each platform |
| "Create a week of content from this podcast transcript" | 5 platform-specific pieces with daily posting schedule |
