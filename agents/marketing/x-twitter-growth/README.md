# Volt - X/Twitter Growth Agent

> AI-powered Twitter/X strategist that writes tweets, threads, and replies to grow your audience as a builder.

## Overview

Volt is your Twitter growth engine, built for founders and builders:

- Writes punchy tweets and high-retention threads
- Plans daily content with optimal posting schedules
- Builds engagement strategy with reply networks
- Tracks impressions, follower growth, and top-performing patterns

## Quick Start

### Installation

```bash
mkdir -p ~/.openclaw/agents/x-twitter-growth/agent
cp SOUL.md ~/.openclaw/agents/x-twitter-growth/agent/

openclaw agents add x-twitter-growth --workspace ~/.openclaw/agents/x-twitter-growth
```

### First Conversation

```bash
openclaw chat x-twitter-growth "Write a tweet about shipping my first SaaS product"
```

## Use Cases

### 1. Daily Tweet Writing
```
You: "Write a tweet about why I prefer Claude over ChatGPT for coding"
Volt: Punchy 280-char tweet with builder angle + posting time
```

### 2. Thread Creation
```
You: "Write a thread about growing from 0 to $5K MRR"
Volt: 7-tweet thread with hook, insights, and CTA
```

### 3. Reply Strategy
```
You: "Write a reply to this viral tweet about AI replacing jobs"
Volt: Sharp reply that adds insight and drives profile visits
```

### 4. Weekly Content Plan
```
You: "Plan my tweets for next week"
Volt: 15 tweets across 5 topic buckets with scheduling times
```

## Tips

1. **Replies are your growth lever** - 70% of follower growth comes from replies
2. **Post at peak hours** - 8-10 AM and 5-7 PM EST for builder audiences
3. **Numbers beat adjectives** - "$3K MRR" outperforms "growing fast"
4. **Ship then tweet** - Past tense builds more credibility than promises
