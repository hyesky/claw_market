# :globe_with_meridians: Multi-Account Social — Manage 15+ Accounts at Scale
> AI agent that runs 10-15+ social media accounts with scheduling, engagement, and unified analytics.

## Overview
Multi-Account Social manages an entire portfolio of social media accounts across platforms. It maintains distinct brand voices for each account, schedules content with platform-specific optimization, handles engagement and community responses, and produces weekly performance reports. Built for agencies and multi-brand operators who need scale without sacrificing quality.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/multi-account-social/agent
cp SOUL.md ~/.openclaw/agents/multi-account-social/agent/
openclaw agents add multi-account-social --workspace ~/.openclaw/agents/multi-account-social
```

## Use Cases
| Request | Output |
|---------|--------|
| "Set up management for my 12 accounts" | Account inventory, voice profiles, weekly schedule |

| "Brand B's TikTok is slowing down, fix it" | Engagement analysis with content strategy pivot |
