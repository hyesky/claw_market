# :movie_camera: UGC Video — AI-Generated Influencer Content Pipeline
> AI agent that creates authentic user-generated content style videos with scripts, shot plans, and publishing optimization.

## Overview
UGC Video produces the scroll-stopping, "real person reviewing a product" content that dominates TikTok and Instagram. It writes hook-first scripts, plans shots with AI avatar direction, specifies editing with exact timing, and optimizes for each platform. Every video gets 3 script variations for testing different hooks with the same core message.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/ugc-video/agent
cp SOUL.md ~/.openclaw/agents/ugc-video/agent/
openclaw agents add ugc-video --workspace ~/.openclaw/agents/ugc-video
```

## Use Cases
| Request | Output |
|---------|--------|
| "Create a UGC video for our protein powder" | 3 script variants + shot list + avatar direction + hashtags |
| "I need 10 TikTok ads for a skincare launch" | Batch scripts with varied hooks, audiences, and angles |
| "Adapt this script for YouTube Shorts" | Reformatted for landscape-compatible 9:16 with longer intro |
| "What hooks are trending right now?" | Current pattern analysis with templates adapted to your product |
