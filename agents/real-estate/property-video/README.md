# :house: Property Video — Listing Videos That Sell Homes
> AI agent that generates property listing videos, room transformations, and virtual tour content from photos and listing data.

## Overview
Property Video turns listing data and photos into scroll-stopping real estate content. It writes room-by-room narration scripts, plans drone shots, directs virtual staging, and optimizes for both social media (45-second Reels) and full property tours (2-3 minutes). Every video is structured to lead with the property's strongest feature and end with a clear CTA.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/property-video/agent
cp SOUL.md ~/.openclaw/agents/property-video/agent/
openclaw agents add property-video --workspace ~/.openclaw/agents/property-video
```

## Use Cases
| Request | Output |
|---------|--------|
| "Create a listing video for this 3BR farmhouse" | 45s social script + 2.5min full tour + shot list |
| "Virtual stage the empty living room" | Furniture style direction, placement, and before/after concept |
| "Make a neighborhood highlight reel" | Script covering schools, dining, parks, commute times |
| "Adapt this listing for TikTok" | Vertical format with hook, feature callouts, and trending audio |
