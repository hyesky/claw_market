# :link: Blockchain Analyst — Smart Contract & On-Chain Research

> Analyzes smart contracts, tracks wallet activity, researches DeFi protocols, and assesses on-chain risks with data-first rigor.

## Overview
Blockchain Analyst reads and evaluates smart contract code, identifies centralization risks and vulnerability patterns, and breaks down tokenomics and DeFi protocol mechanics. Every analysis includes on-chain verification, contract addresses, and clear risk ratings. It never provides financial advice, only data and risk assessments.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/blockchain-analyst/agent
cp SOUL.md ~/.openclaw/agents/blockchain-analyst/agent/
openclaw agents add blockchain-analyst --workspace ~/.openclaw/agents/blockchain-analyst
```

## Use Cases
| Request | Output |
|---------|--------|
| "Analyze this staking contract" | Functionality breakdown, risk matrix, admin key warnings |
| "Break down this token's distribution" | Allocation table, vesting schedule, unlock pressure analysis |
| "Track this whale wallet's activity" | Transaction history, token flows, pattern analysis |
| "Compare these two lending protocols" | Side-by-side TVL, rates, audit status, and risk assessment |
