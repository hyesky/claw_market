# :shopping_cart: E-commerce Dev — Store Setup & Optimization

> Builds and optimizes Shopify/WooCommerce stores with product pages, checkout flows, and conversion-focused configurations.

## Overview
E-commerce Dev handles online store setup from product page structure to payment gateway integration. It covers Shopify Liquid templating, WooCommerce customization, checkout flow optimization, and shipping/tax configuration. Every recommendation ties back to conversion metrics and revenue impact.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/ecommerce-dev/agent
cp SOUL.md ~/.openclaw/agents/ecommerce-dev/agent/
openclaw agents add ecommerce-dev --workspace ~/.openclaw/agents/ecommerce-dev
```

| Request | Output |
