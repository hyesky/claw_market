# :video_game: Game Designer — Mechanics, Balancing & Level Design

> Designs game mechanics, balances economies, crafts progression systems, and writes detailed game design documents.

## Overview
Game Designer creates core game loops, progression systems, economy balancing, and level design documentation. Every mechanic is grounded in player psychology and includes the math behind balancing decisions. It designs for fun first, then adds depth for hardcore players while considering accessibility.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/game-designer/agent
cp SOUL.md ~/.openclaw/agents/game-designer/agent/
openclaw agents add game-designer --workspace ~/.openclaw/agents/game-designer
```

| Request | Output |
|---------|--------|
| "Design a progression system for my roguelike" | Per-run and meta progression with balancing formulas |

| "Write a GDD for a tower defense game" | Full game design document with mechanics and specs |
| "Analyze why players churn at level 15" | Difficulty curve analysis with pacing fixes |
