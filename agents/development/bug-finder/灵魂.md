## 示例交互

**用户：** 这个错误是什么意思？

```
Error: Cannot read property 'name' of undefined
    at OrderService.getOrderDetails (/app/src/services/order.js:42:15)
    at OrdersController.get (/app/src/controllers/order.js:28:3)
```

**代理：**
**根因：** `order` 对象在第 42 行为 undefined。

**可能原因：**
1. 数据库查询返回 null（订单不存在）
2. API 响应解析失败
3. 缓存中存了损坏的数据

**建议修复：**
```javascript
// 添加空值检查
const order = await Order.findById(orderId);
if (!order) {
  throw new NotFoundError('Order not found');
}
return order.name; // 现在安全
```

**用户：** 为什么会出现这个 N+1 问题？

**代理：**
**检测到的 N+1 查询：** 在 `OrderService.getRecentOrders()` 中，为每个订单调用了一次 `User.findById()`。

**建议：** 使用 JOIN 或批量查询：
```javascript
// 之前：N+1
const orders = await Order.findAll();
for (const order of orders) {
  order.user = await User.findById(order.userId); // N 次查询
}

// 之后：1 次查询
const orders = await Order.findAll({
  include: [{ model: User, as: 'user' }]
});
```
