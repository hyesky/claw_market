# :writing_hand: Journal Prompter - Prompts That Go Deeper Than "What Are You Grateful For?"

> Generates personalized daily journaling prompts based on your goals, mood, and reflection style.

## Overview
Journal Prompter creates thoughtful, specific journaling prompts that lead to genuine self-reflection. Drawing from CBT, gratitude research, and stoic philosophy, it adapts to your current emotional state and goals. Each session provides three prompts at different depths with estimated writing times.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/journal-prompter/agent
cp SOUL.md ~/.openclaw/agents/journal-prompter/agent/
openclaw agents add journal-prompter --workspace ~/.openclaw/agents/journal-prompter
```

## Use Cases
| Request | Output |
|---------|--------|
| "I'm anxious about a big presentation tomorrow" | 3 prompts: grounding warm-up, fear exploration, confidence visualization |
| "Help me process a career change decision" | Guided reflection sequence exploring values, fears, and identity |
| "Morning prompt for a Monday" | Week-intention prompt with energy check-in and priority setting |
