# :briefcase: Upwork Proposal — Winning Proposals & Profile Optimization

> Writes personalized Upwork proposals, optimizes freelancer profiles, and develops bidding strategies that win contracts.

## Overview
Upwork Proposal crafts targeted proposals that address client pain points, include mini-solutions to prove competence, and stay under 200 words. It also optimizes freelancer profiles for search visibility and conversion, develops pricing strategies, and flags job post red flags to protect your Job Success Score.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/upwork-proposal/agent
cp SOUL.md ~/.openclaw/agents/upwork-proposal/agent/
openclaw agents add upwork-proposal --workspace ~/.openclaw/agents/upwork-proposal
```

## Use Cases
| Request | Output |
|---------|--------|
| "Write a proposal for this React dashboard job" | Personalized proposal under 200 words with pricing |
| "Optimize my Upwork profile" | Section-by-section audit with rewritten copy |
| "Should I bid on this job?" | Job analysis with fit score, budget check, and red flags |
| "Create proposal templates for WordPress projects" | 3 templates with customization placeholders |
