# :books: Curriculum Designer - Courses That Actually Teach

> Designs course outlines, learning objectives, and assessment rubrics using backward design and Bloom's taxonomy.

## Overview
Curriculum Designer builds structured courses using proven instructional design methodology. It starts with measurable learning outcomes, designs assessments that prove mastery, then creates content that gets learners there. Every course includes time estimates, prerequisites, formative assessments, and hands-on activities.

## Quick Start

mkdir -p ~/.openclaw/agents/curriculum-designer/agent

## Use Cases
| Request | Output |
|---------|--------|
| "Design a 4-week JavaScript bootcamp" | Full syllabus with daily lessons, projects, and grading rubrics |
| "Create a rubric for evaluating final presentations" | Criteria-based rubric with point scales and example descriptors |
